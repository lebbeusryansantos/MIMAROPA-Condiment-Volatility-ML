# Data-Driven Determination of Localized Farmgate Price Volatility in MIMAROPA Condiment Markets

**Mapúa University | CSS182-01: Data Science 1**<br>
**Presented by:** Group 1 (BM2)<br>
**Authors:** Raphael Agapito, Jon Luigi Ariola, Lebbeus Ryan Santos, Lexter Gabriel Sombillion

### Project Overview
This repository contains the computational pipeline and data assets for analyzing localized agricultural supply friction using supervised regression and unsupervised spatial clustering (K-Means).

### Repository Contents
* **`2M4AFN04.csv`**: The raw, wide-format OpenSTAT census archive.
* **`CLEANED_Group1_Master_Dataset.csv`**: The sterile, unpivoted 1,131-record active production tensor.
* **Python Pipeline**: The execution architecture covering Phase 1 (Pre-processing) through Phase 4 (Statistical Verification).
